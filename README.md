# OpenWebUIDemo

## Summary

This is an example configuration of using OpenWebUI

## Getting started

Use [Groq](https://console.groq.com/keys) to get an API key.  Create a file named `GROQ_API_KEY.env`

```GROQ_API_KEY.env
GROQ_API_KEY=<your api key>
```
